# Breadcrumbs
<HeaderComponent htmlTag="studs-breadcrumbs" jsxTag="StudsBreadcrumbs" version="Since 2.0" status="Stable" />

A breadcrumbs is a list of links that help visualize a page's location within a site's hierarchical structure, it allows navigation up to any of the ancestors.

## Default

<ResizerFeatures>
<studs-breadcrumbs separator="/">
  <a href="http://www.strongtie.com">Home</a>
  <a href="http://www.strongtie.com/about">About Us</a>
  <a href="http://www.strongtie.com/contact">Contact</a>
</studs-breadcrumbs>
</ResizerFeatures>
