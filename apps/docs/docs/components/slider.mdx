Sliders allow users to make selections from a range of values.

## Usage

<ResizerFeatures>
  <studs-slider
    label="Peanut Butter"
    rightlabel="Jelly"
    enableinput=""
    enabletooltip=""
    enablelabel=""
  ></studs-slider>
</ResizerFeatures>
