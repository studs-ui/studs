# Card

Cards can be used to group related subjects in a container.

## Card with Button 

<studs-card imageUrl="https://via.placeholder.com/200x200" altText="Studs Card Media" title="Title">
  <div slot="main">This is my custom body content.</div>
  <div slot="footer"><studs-button button-type="primary" onclick="alert('clicked')">Learn more</studs-button></div>
</studs-card>

```html
<studs-card
imageUrl="https://via.placeholder.com/200x200"
altText="Studs Card Media"
title="Title">
  <div slot="main">This is my custom body content.</div>
  <div slot="footer">
    <studs-button button-type="primary" onclick="alert('clicked')">Learn more</studs-button>
  </div>
</studs-card>
```
