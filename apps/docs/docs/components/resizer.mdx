# Resizer

<HeaderComponent
  htmlTag="studs-resizer"
  jsxTag="StudsResizer"
  version="Since 2.0"
  status="Unstable"
  urlGithub="display/resizer.ts"
  urlStrbook="studs-display-resizer--docs"
/>

The Resizer component features two side-by-side panels with the capability for user-driven resizing. It enables intricate layouts by allowing nested instances of Resizer for independent adjustability.

## Usage

<ResizerFeatures>
  <studs-resizer direction="horizontal">
    <studs-resizer-pane size="50">Panel One</studs-resizer-pane>
    <studs-resizer-pane>Panel Two</studs-resizer-pane>
  </studs-resizer>
</ResizerFeatures>

<CodeBlocks
  html={`
<studs-resizer direction="horizontal">
  <studs-resizer-pane size="50">Panel One</studs-resizer-pane>
  <studs-resizer-pane>Panel Two</studs-resizer-pane>
</studs-resizer>
  `}
/>

## Examples

### Horizontal

<ResizerFeatures>
  <studs-resizer
    style={{ height: '300px', display: 'block' }}
    direction="horizontal"
  >
    <studs-resizer-pane>
      <h3 slot="header">Panel One</h3>
      <div slot="content">Padded Content</div>
    </studs-resizer-pane>
    <studs-resizer-pane>
      <h3 slot="header">Panel Two</h3>
      <div slot="content">Padded Content</div>
    </studs-resizer-pane>
  </studs-resizer>
</ResizerFeatures>

### Vertical
