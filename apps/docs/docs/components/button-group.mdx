# Button Group

Button groups can be used to group related buttons into sections.

## Usage

<studs-button-group> 
    <studs-button button-type="primary" btnclasses="-group">Button 1</studs-button>
    <studs-button button-type="primary" btnclasses="-group">Button 2</studs-button>
    <studs-button button-type="primary" btnclasses="-group">Button 3</studs-button>
</studs-button-group>
