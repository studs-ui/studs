# Badge

Badge generates a small badge to the top-right of its child(ren).

## Default

<studs-badge
  icon="info"
  count="1"
  max="99"
  size="medium"
  position="top-right"
  color="primary">
</studs-badge>

```html
<studs-badge
  icon="info"
  count="1"
  max="99"
  size="medium"
  position="top-right"
  color="primary"
>
</studs-badge>
```

## Icon

<studs-badge icon="info" count="0"></studs-badge>

```html
<studs-badge icon="info" count="0"></studs-badge>
```
